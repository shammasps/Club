Prediction Rules

Please read the rules carefully before submitting your predictions.

⚽ Prediction Rules

  • Each user can submit only one prediction per match.
  • Predictions can be edited until 10 minutes before kickoff.
  • After the deadline, predictions are locked automatically.
  • For knockout matches, penalty scores can also be predicted.

⏰ Prediction Deadline

  • Prediction closes 10 minutes before the match starts.
  • Late predictions or updates are not allowed.

🏆 Points System

Prediction Points
Exact Match Score + Exact Penalty Score (if applicable) 7
Exact Match Score 5
Correct Winner 3
Wrong Prediction 0

📌 Important Note

  • For knockout matches decided by penalties, both the regular-time score and the penalty score must be predicted correctly to earn 7 points.
  • If only the regular-time score is correct, you will receive 5 points.
  • If only the match winner is correct, you will receive 3 points.
  • If the prediction is incorrect, you will receive 0 points.

📊 Leaderboard

  • Leaderboard is updated after the administrator publishes the official match result.
  • Rankings are based on total points earned.
  • Correct and wrong predictions are also displayed.

📌 General Rules

  • Only registered users can participate.
  • One prediction is allowed per match.
  • Administrator decisions are final.
  • Official FIFA match results will be used for scoring.
  • All match times are displayed in Indian Standard Time (IST).
